## 함수(function)

### 함수의 정의
  - 다양한 작업을 하나의 단위로 묶어놓은 코드블록

### 함수의 정의 및 사용
1. 함수 선언식

```js
function 함수명(매개변수1, 매개변수2){
  실행문;
  (리턴);
}
```

2. 변수에 함수
  - 변수에 let을 쓰고 함수를 넣으면 순서대로 실행됨 
  - 변수에 함수를 넣기 전에 함수를 사용하면 에러뜸

3. 함수 표현식(arrow function)

4. Function 생성자 함수
```js
let sum = new Function('a', 'b', 'return a + b;');
let sum1 = sum(1,2)
```

### 함수를 사용하는 이유
  1. 코드의 재사용성이 높음
  2. 유지보수성이 높음
  3. 가독성이 높음

### JS 내장 객체
  - 내장 객체에 있는 다양한 함수+ 사용자 정의 함수를 사용하면 가독성이 높아짐

### Object 객체
  - String / Number / Date / Array / Math 등 다양한 객체가 존재함

```js
let person = new Object();
person.fName = '영';
person.lName = '김';

person.getFullName = function(){
  return this.lName + this.fName;
}
console.log(person.getFullName());
```

### String 객체
  1. length
  2. indexOf() - 문자열 내에 특정 문자열이 존재 하는지 찾음 있으면 인덱스 리턴 없으면 -1 리턴
  3. lastIndexOf() 
  4. slice() - 음수 가능
  5. substring() - 음수 불가능
  6. substr() - 요즘은 안 쓰지만 시험문제에 나옴
  7. replace() - 정규식 /영어문자열/i 도 있음
  8. toUpperCas()
  9. toLowerCase()
  10. concat() - 2개 이상의 문자열을 하나로 합치는 함수임 (중요함)
  11. trim() - 공백 삭제 (중요함)
  12. split() - slice()와 비슷한 기능이지만 리턴값이 배열이 나옴

### Number 객체
  1. toString()
  2. toExponential() - 숫자를 지수형으로 반환
  3. toFixed() - 소수점 어떤 자릿수까지 보여줌
  4. toPrecision() - 정수와 소수를 포함해서 몇 번째자리까지 보여줌
  5. parseInt() - 실수를 정수형태로 또는 문자열을 정수형태로 반환
    - NaN은 Not a Number의 축약어임
  6. parseFloat()
  7. Number 객체의 property
    - MAX_VALUE - 최대 수
    - MIN_VALUE - 작은 수
    - POSITIVE_INFINITY - infinity
    - NEGATIVE_INFINITY - -infinity
    - NaN - Not A Number

### Array 객체
  1. toString()
  2. join() - 배열안에 있는 모든 문자와 매개변수를 사용하여 결합함
  3. pop() - 배열에 마지막 데이터를 제거하고 마지막 데이터를 리턴함
  4. push() - 배열에 새로운 요소를 추가
  5. shift() - 첫번째 요소를 제거하고 리턴함 (실무 때 사용)
    - 큐와 스택, 큐를 선인선출(FIFO)
    - FIFO를 구현하기 가장 좋음
  6. unshift() - 배열의 맨 앞에 요소 추가, 배열 길이를 리턴함
  7. 배열의 요소 변경
    ```js
    let number = [1,2,3];
    number[0] = 3; // 0번지가 3으로 바뀜
    ```
  8. splice() - 새로운 요소를 특정 위치에 추가하는 함수 추가 시에 기존 요소를 삭제할 수도 있음
  9. concat() - 두 개의 이상의 배열을 하나로 결합함
  10. slice(p1, p2) 시작값, 한계값
  11. sort() - 문자형 데이터 오름차순으로 정리
    - reverse() - 내림차순 정렬 
  12. filter() - 특정 조건을 만족하는 배열의 요소만 찾아서 새로운 배열로 리턴함 (중요함)
    - 특정 조건을 만족하는 데이터만 추출
    ```js
    배열명.filter(콜백함수(요소[, index[, arr]][, thisArgs]))
    ```
    - 콜백함수 ; 배열 내부의 요소들 하나씩에 적용하는 별개의 함수
    - 처리할 요소
    - index(optional)

  13. map() - 배열에 담긴 Object를 새로운 Object로 변환한 다음에 새로운 배열로 리턴함 (중요함)
    - 
  14. reduce()
