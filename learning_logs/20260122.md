# JS 문법
## 변수와 상수
  - 데이터/객체/함수를 사용
  - 상수(const) : 변화하지 않는 값을 가지는 변수

### 변수 명명 규칙
  1. 문자/숫자/_/$를 포함 할 수 있음
  2. 변수명은 숫자로 시작할 수 없음
  3. JS 예약어는 사용 불가능
  4. 변수명은 대소문자를 구분함

### 선언자의 종류
  1. var : 변수용
  2. let : 변수용
  3. const : 상수용

### var와 let의 차이점
  - 자료형이 고정되어 있지 않음
  - 현재에는 오류를 최소화하기 위해서 let을 사용함
  - 중요한건 var 쓰지 말고 let 쓰자

### const
  - final과 같은 역할
  - Java에서 final을 조심스럽게 사용함
  - JS에서 const을 적극적으로 사용함
  - JS에서는 let을 쓰면 값을 바꿀 수 있기 때문에 const를 써서 동일한 값을 낼 수 있음
  - 함수는 const로 선언함(함수의 결과값을 const에 넣으라는 소리 아님)

# 자료형
## 기본 자료형
  1. String
    - 작은 따옴표를 씀
  2. Number
  3. Boolean
  4. Undefined
    - 데이터 자료형임
  5. Null
    - 데이터 자료형임

  6. Symbol
    - ES6에서 추가됨
  
### Object
  - JS의 모든 것은 객체에 해당함
  - key-value properties로 이루어짐
```js
person["age"] = 51; // 은근 자주 쓰임
person.age = 51; // 이 2개 차이점 엄청 큼

console.log(person['firstName']);
console.log(person.firstName);
```

### Array
  - 배열에 저장된 데이터의 특징은 순서를 가짐
```js
const cars = ['현대', '기아', '혼다'];
console.log(cars[0]);
console.log(cars[1]);
console.log(cars[2]);
cars[2] = '벤츠';
```
