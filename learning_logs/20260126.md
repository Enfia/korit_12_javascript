# JS 주요 웹 API
## LocalStorage / SessionStorage
1. LocalStorage 
  - 저장된 데이터를 삭제하기 전까지 영구히 보존됨 key-value로 이루어져있음
  - key-value는 string임
  - array, object, boolean 등과 같은 string이 아닌 데이터를 JSON으로 저장해야함
  - 데이터를 읽고 싶다면 JSON.parse()를 사용하여 변환해줘야 함
  - LocalStorage는 영구히 저장해도 상관없는 데이터를 저장함
    - 사용자가 마지막으로 보고 있는 화면 url 저장
    - 웹 사이트에서 테마, 개인화 제공하면 해당 정보 저장
  - html 파일도 요구됨
2. SessionStorage
  - localStorage는 지우기 전까지 영구 보존됨
  - sessionStorage는 새로고침하면 다 없어짐
